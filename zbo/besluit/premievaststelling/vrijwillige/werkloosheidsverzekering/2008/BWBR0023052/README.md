##Besluit premievaststelling vrijwillige werkloosheidsverzekering 2008

Het Uitvoeringsinstituut werknemersverzekeringen,  
Gelet op [artikel 74, tweede lid, van de Wet financiering sociale verzekeringen](../../../../../../../wet/wet/financiering/sociale/verzekeringen/BWBR0017745/README.md);
Besluit:    

### Artikel  1  

De premie voor het kalenderjaar 2008 voor de vrijwillige verzekering op grond van de [Werkloosheidswet](../../../../../../../wet/werkloosheidswet/BWBR0004045/README.md) wordt vastgesteld op 4,58%.
