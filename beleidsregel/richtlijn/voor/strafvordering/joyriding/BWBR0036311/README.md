##Richtlijn voor strafvordering joyriding

####Beschrijving

Treedt in werking per 01-03-2015 

Deze richtlijn bevat het strafvorderingsbeleid van het OM inzake overtredingen van artikel 11 van de Wegenverkeerswet 1994.
